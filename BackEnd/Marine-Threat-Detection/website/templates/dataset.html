03 — Annotation Format

Bounding-box labels

Every image is annotated with axis-aligned bounding boxes in YOLO-style format: class cx cy w h, where coordinates are normalized to [0, 1]. Each image gets a sibling .txt file with one line per object.

image_0042.jpg image_0042.txt → 2 0.512 0.337 0.184 0.227 7 0.220 0.711 0.060 0.072
04 — Data Collection

Where images came from

Public marine wildlife and shipping datasets, supplemented with team-captured frames from beach-side webcams. All annotations were manually reviewed in a two-pass labeling protocol — annotator + reviewer — to keep label noise low.
