03 — The Pipeline

Two engines, one Flask app

A custom-trained PyTorch classifier handles single-image verdicts; a YOLOv8 background loop handles real-time webcam surveillance. Both share the same Flask server and the same naval-tactical UI.

  • Classifier: ResNet on 224×224 RGB, 44.8 MB best_model.pth
  • Normalization: mean=[0.284, 0.380, 0.452] std=[0.223, 0.194, 0.188]
  • Live detector: YOLOv8n (COCO-pretrained, threats = boat)
  • Stream: multipart MJPEG over HTTP, ~25 fps cap
  • Auto-snapshots: saved to static/snapshots/ on threat (8 s cooldown)
  • Reports: jsPDF generates a verdict + reasoning + per-class bars on demand
  • Serving: Flask + Jinja2 + vanilla JS, no build step
